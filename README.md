# IT Asset Management

A modern IT Asset Management system for startups and small to medium businesses.

## Project Goals

- Employee Management
- Asset Management
- Device Assignment
- Laptop Tracking
- Hardware Inventory
- Breeze RMM Integration
- Asset Dashboard
- Reports

## Tech Stack

- Frontend: Next.js
- Backend: FastAPI
- Database: PostgreSQL
- ORM: SQLAlchemy
- Authentication: JWT
- Containers: Docker
